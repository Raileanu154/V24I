 BrainPower

 Descriere

BrainPower este un site web pentru un centru de meditații și tutoriat, realizat în cadrul practicii tehnologice.

Site-ul oferă informații despre materiile disponibile, profesori, orar, prețuri și modalități de contact.

 Tehnologii utilizate

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Git și GitHub
- Visual Studio Code

 Paginile site-ului

- Acasă
- Materii
- Profesori
- Orar
- Prețuri
- Contact

 Funcționalități

- meniu responsive;
- quiz demonstrativ;
- calculator de cost;
- validarea formularului;
- slider testimoniale;
- buton Back to Top;
- design adaptat pentru telefon, tabletă și desktop.

 Structura proiectului

brainpower/

- index.html
- subjects.html
- teachers.html
- schedule.html
- prices.html
- contact.html
- css/style.css
- js/main.js
- img/
